### Array.from({伪数组})  将伪数组变成数组，就是只要有length的属性就可以转成数组
```js
let name = "javascript";
console.log(name.length); // 10

let arr = Array.from(name);
console.log(arr); // [ 'j', 'a', 'v', 'a', 's', 'c', 'r', 'i', 'p', 't' ]
```
